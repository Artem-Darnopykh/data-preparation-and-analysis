Опис завдання
Обрано датасет Heart Disease (UCI), який задовольняє всі вимоги: Multivariate, містить Categorical, Integer, Real атрибути, має пропущені значення.

Посилання на датасет: Heart Disease Data Set

Виконано очищення даних (Data Cleaning) та побудовано 6 графіків для аналізу залежностей (lineplot, scatter, гістограма на 5 діапазонів, багатовимірні візуалізації, теплова карта з коефіцієнтами кореляції).

Вимоги до системи
OC: macOS / Linux / Windows

Інтерпретатор: Python 3.9 або вище

Інструкція із запуску та встановлення залежностей
Перейдіть у директорію лабораторної роботи:

Bash
cd lab3
Створіть та активуйте віртуальне середовище:

Bash
python3 -m venv venv
source venv/bin/activate
Встановіть необхідні бібліотеки:

Bash
pip install -r requirements.txt
Запустіть Jupyter Notebook:

Bash
jupyter notebook
Відкрийте файл lab3.ipynb і виконуйте комірки послідовно. Скрипт автоматично перевіряє наявність даних на диску і завантажує їх лише за потреби
