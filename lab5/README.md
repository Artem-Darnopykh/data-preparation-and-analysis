## Опис завдання
Створено інтерактивний веб-додаток за допомогою бібліотеки `Streamlit` для аналізу індексів VCI, TCI та VHI. Реалізовано:

* Двоколонковий макет (панель керування зліва, результати справа).  
* Випадаючі списки (dropdown) для вибору індексу та області.  
* Слайдери для фільтрації за тижнями та роками.  
* Кнопка "Скинути налаштування" (Reset) для повернення до початкового стану через st.`session_state`.  
* Чекбокси для сортування за зростанням/спаданням з обробкою конфлікту (якщо обрані обидва).  
* Три вкладки (Tabs):  
    * Таблиця з відфільтрованими даними.
    * Графік динаміки обраного індексу для обраної області.
    * Графік порівняння середнього значення індексу по всіх областях за обраний період.


## Інструкція із запуску
1. Перейдіть у директорію лабораторної роботи:  
```
cd lab5
```
2. Активуйте віртуальне середовище та встановіть залежності:
```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```  
3. Запустіть Streamlit-додаток:
```
streamlit run app.py
```
