## Опис

Програма захоплює відео з веб-камери та дозволяє застосовувати різні фільтри обробки зображень в реальному часі.

## Вимоги

- **ОС:** Ubuntu 20.04 / 22.04 або сумісний Debian-дистрибутив
- **Камера:** будь-яка веб-камера (USB або вбудована)
- **RAM:** 512 MB і більше

## Залежності

- OpenCV 4.x
- CMake 3.10+
- GCC/G++ з підтримкою C++17

## Встановлення та запуск

```bash
git clone <repo-url>
cd lab6
./preinstall.sh   # встановлення залежностей
./build.sh        # компіляція
./run.sh          # запуск
```

## Керування

| Клавіша | Дія |
|---------|-----|
| `1` | Звичайний режим |
| `2` | Відтінки сірого |
| `3` | Інверсія кольорів |
| `4` | Gaussian Blur |
| `5` | Canny (краї) |
| `6` | Sobel (фільтр) |
| `7` | Glitch ефект |
| `8` | Бінаризація |
| `←↑↓→` | Переміщення хрестика |
| ЛКМ (drag) | Малювання прямокутника |
| `ESC` | Вихід |

**Слайдер Brightness** — регулює яскравість зображення (0-100, середина = без змін).
