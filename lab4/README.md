## Опис проєкту
Створено програму для інтерактивного моделювання та обробки гармонічного сигналу. Програма дозволяє генерує функцію виду y(t) = Asin(ωt +φ), накладає на неї стохастичний шум із заданими статистичними характеристиками та проводить цифрову фільтрацію результату в реальному часі.

## Вимоги до системи

 ОС macOS  Linux  Windows
 Інтерпретатор Python 3.9 або вище

## Інструкція з запуску

1. Перейдіть у директорію лабораторної роботи
```
cd lab4
```
2.Створіть та активуйте віртуальне середовище
```
python3 -m venv venv
venv\Scripts\activate
```

3. Встановіть необхідні бібліотеки
```
pip install -r requirements.txt
```

4. Запустіть Jupyter Notebook
```
jupyter notebook
```
5. Відкрийте файл lab4.ipynb та виконайте код. Скрипт автоматично запустить готовий до роботи графік.
